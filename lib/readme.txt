Compiere2.exe
-------------
Requires that the following environment variables are set:
- JAVA_HOME
- COMPIERE_HOME

Alternatively, you could start Compiere via utils/RUN_Compiere2


Third Party libraries 
----------------------
oracle.jar	- Oracle JDBC 10 (ojdbc14)
sybase.jar	- jTDS
jboss.jar	- JBoss Client files

jPDF.jar	- PDF Process and Viewer from Qoppa Software (demo)
License is included in ComPiere Support.
